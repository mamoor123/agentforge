## 🤔 What is AgentForge?

> **Stop hunting across 50 websites. Find every AI agent in one place.**

AgentForge is the **#1 curated marketplace** for AI agents. We've cataloged **869 real, verified AI tools** across **22 categories** — from Marketing to Coding, Creative to Cybersecurity — so you can discover, compare, and deploy the perfect AI agent for any task.

**Think Product Hunt meets G2 for AI agents.**

## 🚀 Quick Start

### Get running in under 60 seconds

<br />

```bash
# 1️⃣ Clone the repo
git clone https://github.com/mamoor123/agentforge.git
cd agentforge/agentforge

# 2️⃣ Install dependencies
npm install

# 3️⃣ Start development server
npm run dev
```

<br />

Open **[http://localhost:3000](http://localhost:3000)** and start exploring 🎉

## 🌐 Deploy to Vercel

### One-click deploy — zero configuration

<br />

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/mamoor123/agentforge)

Or use the CLI:


## 🤝 Contributing

### We'd love your help making AgentForge better!

<br />

#### Option 1: Submit an Agent 🚀

[Open an issue](https://github.com/mamoor123/agentforge/issues/new) with:
- Agent name
- Website URL
- Category
- Brief description

<br />

#### Option 2: Submit a PR 🔧

```bash
# 1. Fork the repo
# 2. Create your branch
git checkout -b add-new-agent

# 3. Add your agent to src/data/agents.ts
# 4. Commit your changes
git commit -m "Add [Agent Name] to [Category]"

# 5. Push and create PR
git push origin add-new-agent
```

<br />

#### Agent Data Structure

```typescript
{
  id: "your-agent",                    // Unique ID (kebab-case)
  name: "Your Agent",                  // Display name
  tagline: "One-line description",     // Short tagline
  description: "Longer description...", // Full description
  category: "Marketing",               // Category name
  categorySlug: "marketing",           // Category slug
  icon: "🚀",                          // Emoji icon
  rating: 4.5,                         // Rating (1-5)
  users: "100K+",                      // User count
  price: "Freemium",                   // Free | Freemium | Premium | Pay-as-you-go
  featured: false,                     // Featured on homepage?
  new: true,                           // Show "New" badge?
  tags: ["tag1", "tag2", "tag3"],      // Searchable tags
  creator: "Your Company",             // Creator name
  url: "https://youragent.com",        // Website URL
  features: ["Feature 1", "Feature 2"] // Key features list
}
```
